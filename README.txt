Crush!
======

Stereo distortion plugin (VST 32bit, Windows only)

How to build
============

According to the Visual Studio settings, the vstsdk2.4 should be stored in the parent folder to where you save these files.
You will then need to add the 3 C++ src files in the sdk to the project, which will allow the .dll to compile.
